# Color Palette Suggestions for Pharmacy Application

## Option 1: Medical Teal (Recommended)
**Theme:** Professional, trustworthy, healthcare-focused
- **Primary:** Teal/Cyan shades
- **Accent:** Soft coral/salmon for CTAs
- **Neutral:** Cool grays
- **Why:** Teal is associated with healthcare, trust, and professionalism

**Colors:**
- Primary: Teal (#0d9488, #14b8a6, #2dd4bf)
- Secondary: Soft Coral (#f87171, #fb7185)
- Success: Emerald Green (#10b981)
- Warning: Amber (#f59e0b)
- Danger: Rose Red (#f43f5e)
- Info: Sky Blue (#0ea5e9)

---

## Option 2: Modern Indigo ✅ (Now Available)
**Theme:** Modern, tech-forward, professional
- **Primary:** Deep Indigo/Violet
- **Accent:** Warm orange/amber
- **Neutral:** Slate grays
- **Why:** Modern, sophisticated, calming

**Colors:**
- Primary: Indigo (#6366f1, #818cf8, #a5b4fc) - Use: `indigo-500`, `indigo-600`, `indigo-700`
- Secondary: Amber (#f59e0b, #fbbf24) - Use: `amber-500`, `amber-400`
- Success: Emerald (#10b981) - Use: `emerald-500` or `green-500`
- Warning: Yellow (#eab308) - Use: `yellow-500`
- Danger: Red (#ef4444) - Use: `red-500`
- Info: Cyan (#06b6d4) - Use: `cyan-500`

**Usage Examples:**
```jsx
// Primary buttons
<button className="bg-indigo-600 hover:bg-indigo-700 text-white">Button</button>

// Accent highlights
<span className="text-amber-500">Highlight</span>

// Cards with indigo theme
<div className="bg-indigo-50 border-indigo-200">Card</div>
```
